# FLYING LIQUOURICE

## Setting up

* `composer install`
* `./vendor/bin/phinx migrate -e development`

## Running

* Set your ip and port in ./bin/battleship.php
* `php ./bin/battleship.php`
* connect to your ip and port using telnet: `telnet [ip] [port]`
