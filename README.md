# FLYING LIQUOURICE

## Setting up

* `composer install`
* `./vendor/bin/phinx migrate -e development`

## Running

* run `php ./bin/battleship.php [ip] [port]` ([ip] and [port] are optional and default to 127.0.0.1 and 1337)
* connect to your ip and port using telnet: `telnet [ip] [port]`
